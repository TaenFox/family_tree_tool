= Шаблоны карточек

Этот каталог содержит минимальные шаблоны карточек по правилам картотеки.

Шаблон копируется целиком в нужный раздел:

* `person-card/` - в `03-people/`;
* `group-card/` - в `04-groups/`;
* `place-card/` - в `05-places/`;
* `source-card/` - в `06-sources/`;
* `research-card/` - в `07-research/`.

После копирования нужно:

* переименовать каталог по номеру карточки;
* заполнить `card.adoc`;
* добавить изображения в `images/`, если они есть.
